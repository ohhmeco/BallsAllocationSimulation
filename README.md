# Clone the Repository

Clone the project from your GitHub repository:

> git clone [https://github.com/ohhmeco/BallsAllocationSimulation](https://github.com/ohhmeco/BallsAllocationSimulation.git)

> cd GaltonBox-simulation

# Create a Virtual Environment 

> python3 -m new_venv venv

# Activate the environment (macOS/Linux)
> source venv/bin/activate

# Install Dependencies
> pip install -r requirements.txt

# Run the Simulation

> python proj.py
